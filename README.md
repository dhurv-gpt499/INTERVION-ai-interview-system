## What is INTERVION?

INTERVION is a fully local, real-time AI interview simulator built for developers and students preparing for technical interviews.

It doesn't just ask questions — it **listens actively**, **detects how you feel**, and **adapts in real time**.

- Missed a beat? It comforts you.
- Answered confidently? It goes deeper.
- Froze mid-sentence? It gives you space.

Everything runs **locally on your machine** — no cloud APIs, no data leaving your system, no subscriptions.

---

## Features

- 🎙️ **Real-Time Speech Recognition** — Whisper-powered STT with Hinglish support
- 🧠 **Intelligent Questioning** — Resume-aware questions generated by Qwen 2.5 7B
- 💬 **Streaming AI Responses** — LLM output fed token-by-token directly into TTS
- 🎧 **Natural Voice Output** — Professional AI interviewer voice via Edge-TTS
- 👁️ **Emotion Detection** — Real-time anxiety and confidence scoring from webcam
- ⏸️ **Pause-Aware Pipeline** — VAD detects hesitation, silence, and answer completion
- 📊 **Post-Interview Feedback** — Detailed scoring, strengths, weaknesses, improvement areas
- 🔄 **Cross-Session Memory** — Tracks progress across multiple interviews, never repeats questions
- 🗂️ **Resume Parsing** — Automatic extraction of skills, projects, and experience from PDF
- 🔒 **100% Local** — All models run on your GPU, your data stays on your machine

## Tech Stack

| Layer | Technology | Purpose |
|---|---|---|
| **STT** | faster-whisper (small) | Real-time speech to text |
| **VAD** | Silero VAD | Voice activity + pause detection |
| **LLM** | Qwen 2.5 7B Q4 via Ollama | Interview intelligence |
| **TTS** | Edge-TTS | Natural streaming voice output |
| **Vision** | OpenCV + MediaPipe | Webcam + face analysis |
| **RAG** | LangChain + FAISS | Resume-aware context retrieval |
| **DB** | SQLite | Session + progress storage |
| **Audio** | PyAudio + pygame | Mic capture + audio playback |

**VRAM Budget (RTX 3050 6GB)**
```
faster-whisper small  →  ~1.0 GB
Qwen 2.5 7B Q4_K_M    →  ~4.3 GB
Vision (CPU)          →   0.0 GB
TTS (CPU)             →   0.0 GB
Total                 →  ~5.3 GB ✅
```

## Installation

> **Only requirement: Python 3.11** — everything else installs automatically.

```bash
# 1. Clone the repo
git clone https://github.com/yourusername/INTERVION.git
cd INTERVION

# 2. Run the auto-installer (does everything below automatically)
python install.py
```

**What `install.py` does automatically:**
- ✅ Creates Python virtual environment
- ✅ Downloads and installs Ollama silently
- ✅ Pulls Qwen 2.5 7B model (~4GB, one time only)
- ✅ Installs all pip dependencies (PyTorch CUDA, Whisper, etc.)
- ✅ Initializes the database

```bash
# 3. Start the app
python run_model.py
```

## How It Works

```
1. Upload your resume (PDF)
        ↓
2. Qwen parses resume → extracts skills, projects, experience
        ↓
3. System prompt built with your full profile
        ↓
4. Interview begins — AI asks tailored question via TTS
        ↓
5. You answer via microphone
        ↓
6. Whisper transcribes in real-time chunks
        ↓
7. VAD detects answer completion (3s pause)
        ↓
8. Qwen evaluates answer → streams follow-up/next question
        ↓
9. TTS speaks response sentence by sentence
        ↓
10. Vision pipeline tracks anxiety + confidence throughout
        ↓
11. Post-interview: detailed feedback report generated
```
